# Command: execute-prp

## Goal
Implement the PRP with validation.

## Loop
1. PLAN: tasks, files, modules
2. IMPLEMENT: code, tests, docs
3. VALIDATE:
   - Build: xcodebuild
   - Unit/UI tests
   - SwiftLint
   - A11y check (labels)
   - Perf smoke
   - Privacy scan
4. FIX on failures, repeat
5. OUTPUT: summary per gate
